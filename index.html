Ready to explore Seattle's Open Data Atlas?

This project was created for the WAGISA Map Contest and inspired by the Dev Summit and the modernization of Seattle Public Utilities' Utiliview Web Mapping Application. It reimagines how we browse, search, and interact with geospatial data—using Vite, ArcGIS Components, and the ArcGIS Maps SDK to build a fast, lightweight, fully customizable web app.

Seattle GeoData Explorer was built entirely from scratch from a single CSV export of the Seattle GIS Open Data site:
https://data-seattlecitygis.opendata.arcgis.com/

That CSV contained the complete dataset catalog, which now powers the application's dynamic layer list, metadata browser, filters, and in-app layer loading.

With the help of AI as a force multiplier, the app was built over three focused evenings. It's open source, experimental, and a little rough around the edges—but it demonstrates a novel approach to interacting with data repositories and building custom ArcGIS-powered interfaces quickly and efficiently.

Use the catalog to filter by type, category, source, and tags. Add active layers, inspect metadata, and open attribute tables—all with one click.
